## ✨ **코딩하면서 어려웠던점**

> ### 처음에는 하나의 파일에 코딩을 하고 나중에 class를 사용해 모듈화를 진행했다.
>
> ### 나름 구조화를 잘 했다고 생각했으나, 모듈로 바꾸려 하니 기능별로 분류가 제대로 안되는 문제가발생했다.
